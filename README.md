**Ванильная сборка Minecraft-сервера на Fabric — только оптимизация и удобство, без изменений геймплея.**

Сами моды в репозиторий не загружены.

---

## 🟠 Почему Fabric?

Paper и другие сервера на Bukkit API, они меняют поведение игры (редстоун, спаун мобов, механики), что ломает ванильный опыт.  
**Fabric** сохраняет оригинальную логику игры, добавляя только то, что нужно для производительности.

Эта сборка даёт **100% ванильный геймплей** с улучшенной производительностью.

## ⚡ Производительность

Протестировано на:
- **Процессор:** Ryzen 5 5600G (4.7 ГГц)
- **Оперативная память:** 8 ГБ (сервер потреблял 4–5 ГБ)

- **Результат:** стабильная работа при **30 одновременных игроках**.
- **Есть Возможность:** получить стабильные 30 - 50 онайна, уменьшив spawn rate до 50-70
- /servercore settings mobcap_percentage <vaule>
