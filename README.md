Real-Vanilla Server Pack

Ванильная сборка Minecraft-сервера на Fabric
🟠Paper и другие сервера на Bukkit API меняют поведение игры (редстоун, спаун мобов, механики), что ломает ванильный опыт. 
Поэтому за основу взят Fabric — он сохраняет оригинальную логику игры, добавляя только то, что нужно для производительности.

🤔Почему Fabric?
Paper и его форки изменяют ванильную механику — редстоун работает иначе, спаун мобов отличается. 
Эта сборка даёт 100% ванильный геймплей с улучшенной производительностью.

⚠️Сборка работает полностью на серверной стороне, качать моды другим ненужно !

📦 Что внутри
🛡️ Защита и безопасность
EasyAuth — Регистрация игроков, защита аккаунтов
Antixray — Защита от читеров с рентген-паками
Ledger — Полный лог действий игроков (аналог CoreProtect)
LuckPerms	— Гибкая система прав (Игрок, Владелец и др.)

⚙️ Администрирование и управление
CustomMotd — Красивое сообщение при подключении к серверу
TAB	— Минималистичный и настраиваемый TAB-лист
TabTps — Команда /tabtps для мониторинга производительности
Spark	— Отладка и сбор данных о работе сервера
Placeholder API	— Плейсхолдеры для всех плагинов

🚀 Оптимизация производительности
C2ME — (Concurrent Chunk Management Engine)	Многопоточная загрузка чанков — мощный прирост производительности без изменения генерации мира
Lithium	— Оптимизация игровой логики и механик
FerriteCore	—Снижение потребления оперативной памяти
ServerCore —Оптимизация работы сервера
Krypton	—Оптимизация сетевого кода
VMP (Very Many Players)	—Оптимизация для большого количества игроков
Noisium	— Оптимизация генерации шума (шумов Perlin/Simplex)
ScalableLux	— Оптимизация освещения (Fork Starlight)

🎭 SMP / RP — Взаимодействие и атмосфера
VoiceChat	— Голосовой чат с приближением (как в SCP или Rust)
EmoteCraft — Анимации и эмоции для персонажей
FSit — Возможность садиться на ступеньки, блоки и т.д.
PlayerAnimationLib — Библиотека анимаций для других модов
DynamicLights	— Динамическое освещение — предметы в руке светятся
SkinRestorer — Принудительная установка скинов (особенно для игроков без лицензии)

📦 Библиотеки и зависимости
Fabric API — Основа для работы всех Fabric-модов
Fabric Language Kotlin — Поддержка модов, написанных на Kotlin
Fabric ORM Jimmer — Библиотека для работы с базами данных

🗺️ Мир и генерация
Chunky — Предварительная загрузка чанков — плавный полёт без подгрузок
